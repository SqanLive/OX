## OX
Classic 2 player tictactoe game.  

Program will ask for an input each turn.  

## Example:
   ---------  
3  |       |  
2  |       |  
1  |       |  
   ---------  
    1  2  3  
>Enter the cordinates:   
 
input> 3 2  

   ---------  
3  |       |  
2  |     X |  
1  |       |  
   ---------  
    1  2  3  
>Enter the cordinates:  

input> 1 3
